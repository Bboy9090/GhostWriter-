## 🎯 Key Usability Features

### 1. **Rich Tooltips Everywhere** 🏷️
- **Portal Logo**: Hover to see status and instructions
- **All Buttons**: Contextual help on every control
- **Keyboard Shortcuts**: Shown in tooltips for discoverability
- **Smart Positioning**: Tooltips adapt to available space

### 2. **Keyboard Shortcuts** ⌨️
Full keyboard control for power users:
- **`Space`** - Toggle capture on/off
- **`← → ↑ ↓`** - Move portal (10px steps)
- **`Shift + Arrows`** - Fast move (20px steps)
- **`Esc`** - Minimize portal
- **`H`** - Toggle help panel

**Smart Detection**: Shortcuts only work when not typing in inputs!

### 3. **First-Time Onboarding** 🎓
- **Welcome Message**: Appears on first use

## 🎮 User Experience Flow

### First-Time User
1. **Portal appears** with welcome message
2. **Reads instructions** in onboarding card
3. **Clicks "Got it!"** to dismiss
4. **Hovers over portal** to see tooltip
5. **Clicks logo** to activate
6. **Sees success toast** confirming action
7. **Learns shortcuts** from help panel

### Returning User
1. **Portal remembers position** from last session
2. **Uses keyboard shortcuts** for speed
3. **Hovers for quick info** when needed
4. **Minimizes when not needed** (Esc key)
5. **Restores easily** when needed

### Power User
1. **Uses Space** to toggle quickly
2. **Uses Arrow keys** to position precisely
3. **Uses Shift+Arrows** for fast movement
4. **Presses H** for help when needed
5. **Keyboard-only workflow** for efficiency
